```yaml

kind: ReplicaSet
```

## PART 3 - Deployment

Create a new deployment named `api-deployment` with the image `registry.cluster.wik.cloud/public/echo` for update strategy use `RollingUpdate`.

### Configuration file:
```yaml
apiVersion: apps/v1
kind: Deployment
metadata:
  name: api-deployment
spec:
  replicas: 4
  selector:
    matchLabels:
      app: api
  template:
    metadata:
      labels:
        app: api
    spec:
      containers:
        - name: api-container
          image: registry.cluster.wik.cloud/public/echo
  strategy:
    type: RollingUpdate
    rollingUpdate:
      maxUnavailable: 50%
```

## part 4 - Service

1. Add service to `replicaset-api-pod.yaml`:
```yaml
---
# service for api-replicaset pods forwarding traffic to port 8080
apiVersion: v1
kind: Service
metadata:
  name: api-service
spec:
  selector:
    app: api
  ports:
    - protocol: TCP
      port: 8080
      targetPort: 8080
  type: ClusterIP
```

## part 5 - Ingress

    ```yaml
    apiVersion: networking.k8s.io/v1
    kind: Ingress
    metadata:
      name: echo-ingress

        - host: rust-api.local
          http:
            paths:
              - path: /
                pathType: Prefix
    ```
